将本月各天的工作输出为以下表格形式：

| 日期       | 项目      | 领域 | 任务类型       | 工时 | 任务详情            |
| ---------- | --------- | ---- | -------------- | ---- | ------------------- |
| 2026-05-01 | Project 1 | 软件 | 代码编写与调试 | 8    | "1. xxx<br/>2. xxx" |
| 2026-05-02 | 休假      |      | 休假           | 8    |                     |
| 2026-05-03 | 休假      |      | 休假           | 8    |                     |
| 2026-05-04 | Project 2 |      | 代码编写与调试 | 8    |                     |
| 2026-05-05 | Project 2 |      | 代码编写与调试 | 8    |                     |

- 日期唯一，每天只能有一条记录
- 通常情况下，周末、节假日是休息：项目=休假，领域=空，任务类型=休假，工时=8，任务详情=空
- 任务类型：['DA', '数据闭环', '数据生产', '数据运营', '实验室运营', '车辆运营', '需求管理', '架构设计', '概要设计', '详细设计', '代码编写与调试', 'bug fix', '测试case设计', '测试case编写与调试', '测试', '需求评估与分析', '质量管理', '市场管理', '客户管理', '销售管理', '项目管理', '评审']
- 任务详情：使用 "" 包裹内容，可以包含 `<br/>` 换行符
